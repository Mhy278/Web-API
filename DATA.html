Web-API Additional Data

The Web-API provides additional data which is not documented on the endpoints because it may or may not be present.

The following documentation tries to list all these properties, they will be present on objects for which it makes sense. The current implementation state of the various data can be found here.

Objects from the /player, /entity, /tile-entity as well as any item stacks (in their data properties) may contain none, one or multiple of these properties.

In the documentation below the "carrier" of a property is often referred to as the 'object', this is done to emphasize the fact that the property can be attached to possibly various different things.

Achievements

Contains a list of achievements that have been granted to the object. json "achievements": [ { "id": "mine_wood", "name": "Getting Wood", "class": "net.minecraft.stats.Achievement", "description": "Attack a tree until a block of wood pops out" } ]

Age

Contains information about the age of an object. json "age": { "adult": false, "age": 35, }

Career

Describes the profession of the object. json "career": "minecraft:priest"

Durability

Describes the durability of an object. json "durability": { "unbreakable": false, "durability": 1561 }

Dye

The color of an object. json "dye": "white"

Experience

The amount of experience an object has. json "experience": { "level": 0, "experience": 0, "totalExperience": 0 }

Food

The current food levels of an object. json "food": { "foodLevel": 20, "exhaustion": 3.049999952316284, "saturation": 1.0 }

GameMode

The game mode the object is currently in. json "gameMode": "minecraft:creative"

Health

The health information of the object json "health": { "current": 20.0, "max": 20.0 }

Joined

Information about when the object first/last joined the server. json "joined": { "first": 1489329688, "last": 1492426023 }

PotionEffects

A list of effects from potions attached to/affecting the object. json "potionEffects": [{ "type": "minecraft:instant_damage", "amplifier": 1, "duration": 1 }]

Sheared

True if the object has been sheared, false otherwise. json "sheared": false

Sign

A list of lines of text that are displayed on an object. json "sign": [ "Hi", "How are you?", "", "Web-API" ]

Spawn

Information about which entity is spawned by the object. json "spawn": "minecraft:skeleton_horse"
